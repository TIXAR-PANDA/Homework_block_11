NumPy (расшифровывается как Numerical Python) — это библиотека для Python, которая позволяет
работать с многомерными массивами и матрицами. Она подходит для научных и математических расчётов,
поскольку отличается быстротой и эффективностью.

    Библиотека pandas предназначена для анализа уже структурированных данных,
то есть размещённых не хаотично, а в таблицах. Такие данные ещё называют «панельными»,
откуда и пошло название библиотеки: PANel DAta → pandas.
Библиотека pandas в Python нужна потому, что структурированные данные часто бывают в
форматах, которые не подходят моделям машинного обучения, — например, в виде таблиц Excel.
Pandas позволяет группировать данные, генерировать сводные таблицы, вычислять статистики
и рисовать графики.
    Pandas предоставляет два типа классов для обработки данных:
1. Series: одномерный помеченный массив, содержащий данные любого типа,
такие как целые числа, строки, объекты Python и т.д.
2. DataFrame: двумерная структура данных, которая хранит данные в виде двумерного массива
или таблицы со строками и столбцами.

    Python Requests — это библиотека, позволяющая отправлять HTTP-запросы разного уровня
сложности веб-сервисам и получать от них соответствующие ответы. При помощи этой библиотеки
можно, например, получить прогноз погоды на завтрашний день для любого города.
    response.status_code в Python возвращает числовое значение,
указывающее на статус запроса. Например, статус 200 (OK) значит, что запрос успешно выполнен,
а статус 404 (NOT FOUND) говорит о том, что запрашиваемый ресурс не был найден.
    data = response.json()
    Метод response.json() в Python возвращает JSON-объект полученного результата.
Для использования этого метода результат должен быть в формате JSON, в противном случае
метод выведет ошибку. После выполнения response.json() получается словарь, который можно использовать
так же, как любой другой словарь в Python.
